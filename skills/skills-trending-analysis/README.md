# Skills Trending Analysis

This is a Skill that fetches the trending page from skills.sh and generates statistics and trend summaries for skills, developers, and keywords.

## Purpose

- Fetch the popular skills ranking
- Aggregate by keyword and developer
- Calculate ecosystem concentration metrics
- Generate analysis and summary reports via AI

## Prerequisites

- Python 3.10+
- Playwright (Chromium)

Dependencies are only installed in the `.venv` within this Skill's directory.

## Quick Start

```bash
# First time only: Setup virtual environment and Chromium
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
playwright install chromium

# Fetch data (without keyword)
python3 scripts/fetch_trending.py

# Statistical analysis
python3 scripts/analyze_trending.py --input tmp/trending.json --output tmp/trending_analysis.json
```

After that, the AI refers to `tmp/trending_analysis.json` and generates a report according to `references/output-format.md`.

## Output

- **tmp/trending.json** — The fetched list of skills (title, developer, installs)
- **tmp/trending_analysis.json** — The statistical results (summary, skill_ranking, keyword_ranking, developer_ranking, concentration)
- Report — Trending summary / Top skills / Keyword ranking / Developer ranking / Ecosystem analysis
